About This Tool

A small public-good utility to help users understand Tor network conditions.

What this site does?

  • Uses public Tor network metadata
  • Analyzes relay availability across regions
  • Shows trends to help decide when to retry

What this site does?NOT do

  • Does not browse or index onion sites
  • Does not track users or store IP addresses
  • Does not recommend exit nodes or routes

Privacy & Ethics

This site uses no cookies, no analytics, and no tracking of any kind. All processing is anonymous and based solely on public Tor metadata.
